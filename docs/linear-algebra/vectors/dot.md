**Routine Name:** dot

**Author:** Jake Epperson

**Language:** Python

**Description/Purpose:** Computes the dot product of two vectors of appropriate size

**Inputs:**

- u : numpy array (vector)
- v : numpy array (vector)

**Output:** u $\cdot$ v : number

**Usage/Example:**
